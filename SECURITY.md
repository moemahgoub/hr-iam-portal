# Security Policy

## Supported Versions
This is an educational/demo project. It is provided "as-is" without warranties.

## Reporting a Vulnerability
If you find a security issue, please do not open a public GitHub Issue.

Instead:
- Use GitHub “Security” tab → “Report a vulnerability” (if available), or
- Contact the repository owner privately.

## Sensitive Data
Please do not share:
- Access tokens
- Client secrets
- Passwords
- Tenant IDs, app IDs, object IDs
- Public IPs or private endpoints

If you accidentally posted sensitive data, delete it immediately and rotate the secret in your identity provider.
